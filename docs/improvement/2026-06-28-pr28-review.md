## Architecture (high ‚Üí low)

- **[high]** `core/internal/manager/session.go:147-164` ‚Äî `sessionCreateRun` switches on
  `home.Type` but calls a single `channelAdmin` resolved by `firstAdmin` (first gateway
  exposing Admin, registration-order dependent). With both Discord + terminal admins present,
  a `terminal` home can be minted by Discord (or vice-versa). Select the admin matching the
  home's gateway kind.
- **[med]** `session.go:148,173-176` ‚Äî gateway-kind string `"terminal"` and Discord
  `<#%s>` vs terminal output branching leak into platform-agnostic core (OCP). Let the
  home/gateway descriptor render its own channel reference.
- **[med]** `terminal.go:85-104` + `serve.go` ‚Äî TUI‚Üîhost SessionControl handshake busy-polls
  every 50ms up to a magic 5s deadline; no readiness signal. Slow start ‚Üí default session
  silently never appears. Use a ready channel.
- **[med]** `terminal.go:76,247-272` ‚Äî terminal drives sessions via stringly-typed argv and
  flag string-matching (`--terminal_only`, `terminal_only:`). A manager flag rename breaks it
  silently. Prefer a structured SessionControl API.
- **[low]** `terminal.go:189-192` ‚Äî `Menu` drops `[]contracts.Choice`; host Pick routing is
  unreachable from the TUI ‚Üí a Menu turn hangs. Render choices or declare unsupported.

## Performance

- **[med]** `tui/tui.go:260-271` ‚Äî `syncViewport` does `strings.Join(tb.lines, "\n")` on every
  event; `tb.lines` is unbounded ‚Üí O(n¬≤) CPU + alloc churn over a session (worst on streaming
  chunks). Cap to a ring + incremental render.
- **[med]** `tui/tui.go:50-58` ‚Äî `tab.lines` append-only, never trimmed ‚Üí unbounded memory per
  tab for daemon lifetime. (Same root cause; also flagged under security.)
- **[low]** `tui/tui.go:402-404` ‚Äî 1s tick unconditionally re-enumerates sessions + rebuilds
  maps even when idle. Skip when unchanged.
- **[low]** `terminal.go:46-53` ‚Äî single `mu` serializes all per-session Read polling
  (N locks / 50ms). Shard if session counts grow. (No leak/deadlock.)

## Security (all low/med ‚Äî local operator surface)

- **[med]** `terminal.go:271` ‚Äî `Dispatch` uses `context.Background()` and runs synchronously
  in the Bubbletea Update loop. A slow `/session create --clone` (10-min `cloneTimeout`) freezes
  the whole TUI uncancellably. Run off the Update goroutine + thread ctx.
- **[med]** `tui/tui.go:227` ‚Äî unbounded per-tab transcript fed by high-volume agent output
  (memory growth). Cap line count.
- **[low]** `session.go:46` ‚Äî no cap on concurrent sessions; each create spawns a worktree +
  child process. A scripted loop exhausts resources. Add a max-session ceiling.
- **[low]** `tui/tui.go:171` ‚Äî TUI forwards any `/`-verb to the full dispatcher; no allow-list.
  Not injection (tokenized), but future destructive verbs become reachable. Gate to session verbs.
- **[low]** `terminal.go:209` ‚Äî `emit` drops events on full 256-buffer; dropped `closed`/`reply`
  events affect control state (see bugs).

## Bugs

- **[med]** `tui/tui.go:260` ‚Äî `syncViewport` unconditionally `GotoBottom()` on every event for
  the active tab; the active tab is usually the streaming one ‚Üí PgUp/PgDn scrollback (advertised
  in README) is yanked back to bottom mid-turn. Only auto-scroll when already at bottom.
- **[med]** `terminal.go:209` ‚Äî if the final `reply{Done:true}` is dropped under backpressure,
  the tab's `busy`/`‚ü≥ working` state never clears and reply text is lost. Clear busy defensively.
- **[med]** `tui/tui.go:201` ‚Äî `confirmClose` closes by `tb.label`, which defaults to the channel
  id and is only upgraded to the session name on the 1s tick. Ctrl+W+y in that window dispatches
  `--name <channel-id>`, which fails silently (error discarded `_, _ =`). Close by resolved name +
  surface the error.
- **[low]** `session.go:213` ‚Äî `/session list` always formats `<#%s>` Discord mention syntax;
  the terminal-neutral branch added for create was not applied to list ‚Üí raw `<#terminal/...>`
  leaks into the TUI.
- **[low]** `tui/tui.go:171` ‚Äî slash-command output/errors written to `m.tabs[m.active]` guarded
  by `if tb != nil`; before any tab exists, failures (e.g. invalid `/session create`) are invisible.
- **[low]** `tui/tui.go:425` ‚Äî help overlay makes View() render Height+1 lines (vp height not
  reduced for the 3-line help block) ‚Üí tab bar pushed off / flicker.
